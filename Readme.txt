1.	Installation

Simply follow the instructions given by the InstallShield setup program.

Do not hesitate to forward this software within your company site. But do not copy
or redistribute this software to another site or another company, as you need the
written consent of Si Automation to do so. Simply contact us for new copies that we
will be pleased to send to you in the shortest time.

The installShield will install on your computer the following files:
	AreYouThere 1.1 User Guide.pdf		User Manual
	AreYouThere.exe				The AreYouThere Executable file
	AreYouThere.ini				Default Start Configuration file
	DefaultLibrary.sil			Standard library of SECS messages
	Readme.txt				This file
	_ISREG.DLL				InstallShield DLL
	_DEISREG.ISR				Install information to uninstall, do not remove
	DeIsL1.isu				Install information to uninstall, do not remove

Tips:	- If you install AreYouThere 1.1 Freeware as an administrator,
you will have the AreYouThere shortcuts in the Start Menu even if you
log as a simple user. On the opposite, you will not have those shortcuts
when you log on as an administrator if the install has been done as a
simple user.
	- If you install AreYouThere 1.1 Freeware as an administrator,
then you will not be able to desinstall it as a simple user.



The Uninstallation of AreYouThere 1.1 is easily reachable through the “Add/Remove Programs”
applet in the Control Panel of Windows.
